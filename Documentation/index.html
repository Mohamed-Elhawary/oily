1- Install Smart Editor

Code Editor is the way and the gate that will let you have full control on your website, so you may need to install a smart editor that can be helpful for you when you treat with the website pages, one of the most powerful editors in the market is Visual Studio you can install it now it is free.

2- Open the Project Folder Workspace

After installing the Visual Studio editor, open it and in the top bar click on file menu then select Open Folder, then select the project folder to open all its structure as a workspace, dont't forget to launch the Auto Save option in the same file menu as shown in the image below to save your changes to the project automatically.

3- Open the JSON File for the Products

Open the file that named products.json to can delete, add or edit an product as you want

2- Be Familier with a Product Component

Each product has a component of data that belongs to it only, tha heading names of the data in each component are similar, the difference is in the value.

Let's take a look what each product contains:-
- id: it is a unique number for the product.

- title: the name of the product.

- brand: the product brand (mobil, total, shell,..)

- oilType: the oil type of the product (diesel or gasoline)

- description: the description of the product name in its full name, prefers to write it inside brackets like the image below.

- descriptionID: it is ike the description of the product exactly but it is written in a different shape, between each two words add (_) sign and all letters in samll case, this kind of data is important and must be added like the way i explained because javascript part depends on it to work your website without any problems.

- priceAfter: it is the price of the product after discount, write it as a number without any kind of currency like L.E or USD, just the price number only like the image below.

- priceBefore: it is the price of the product before discount, write it as a number without any kind of currency like L.E or USD, just the price number only like the image below.

- stars: you have here two options of stars, products has 3 stars so write the value as (3), or a product has 2 stars so write the value as (2).

- images: this data is very important and sensitive, so be aware when you edit or add it, this is the (url) of the product image, each product has its unique (url), i will explain later in the next steps how to make your new product url image.

Note that each product is grouped together with its siblings products which have the same brand name, you may notice for example that mobil brand has its products that are grouped together inside the same square brackets in one collection, this note is very important to take care of it when you add a new product while you MUST know before anythig what is the brand name of your product.

4- How to add a new Product in an already existed Oil Brand

First you must know the product brand name which will be one of those (Mobil, Total or Shell), so go to the this brand collection in the json file and make a new product component inside this brand collection and this component will have the same stracture as the other products inside this brand collection.

Second add your new product data details as i showed before, it must have a unique (id) number, so give it a unique (id), then add the other values of the data like the product name, description, price, and so on.

Third when you reach to the (image) data, you must be aware here, you need to get a suitable image for the product that refers to it, prefers to be in size 228px*228px, then save this image in the tanks folder inside the images folder and give it a name like tank12 but the number of the this tank must be unique also, while if there is another tank that was named (tank12) so you may need to name your product image with another tank name for example (tank13) and so on.

Fourth after you named the image, go to the json file again and add the url of this image as a value for the image data in your new product component, the url will be in this shape images/tanks/ImageName.imageExtension for example (images/tanks/tank13.png)

Now your product is ready, you don't need any more steps to do, you just now add a new product to a specific existed oil brand with a few steps, and the product will be dynamically added to the website after the auto save is runned.

Hint1: Note that each value of the data must be written inside double quotes "" in the json file.

Hint2: to show a live preview of your website to see the effects of your changes once you made it, you can go to the index.html file in the website workspace and click a right click on it then choose Open with live server option in the context menu that will be shown to you, then the website will be opened in your default browser, while you will have a full control to transport between its pages smoothly and see the changes.
